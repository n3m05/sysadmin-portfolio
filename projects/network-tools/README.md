# Network Diagnostics Toolkit

This Python script performs simple network diagnostics useful for sysadmins.

## Features
- Ping a list of hosts
- Check open TCP ports
- Easy to extend for traceroute or logging

## Installation
```bash
git clone https://github.com/n3m05/sysadmin-portfolio.git
cd sysadmin-portfolio/projects/network-tools
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

## Usage
python net_diag.py
